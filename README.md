# colorectal-cancer-biomarkers
Computational Analysis of HCFC2 Knockdown Transcriptomics in Colorectal Cancer Cell Lines
Raw gene count data (GSE335441) was obtained from the NCBI Gene Expression Omnibus. The dataset comprised 18 RNA-seq samples from two colorectal cancer cell lines (SW620 and T84) subjected to shRNA-mediated HCFC2 knockdown (constructs #287 and #449) or a non-targeting control, with three biological replicates per condition. Data were processed in Python using pandas and NumPy. Counts were normalised to log₂ counts per million (CPM) after removal of zero-count genes. Principal component analysis (PCA) was performed on the top 5,000 most variable genes using scikit-learn. Differential expression between knockdown and control samples was assessed within each cell line using Welch's t-test on log₂ CPM values, with Benjamini–Hochberg correction for multiple testing. Genes with |log₂FC| ≥ 1 and FDR < 0.05 were considered differentially expressed. Ninety-nine genes were consistently dysregulated across both cell lines (75 downregulated, 24 upregulated), including the epithelial marker CDH1 and the actin remodelling genes WASL and ARPC3, suggesting a role for HCFC2 in maintaining epithelial identity in colorectal cancer. All visualisations were generated using matplotlib and seaborn.
